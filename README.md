# Motorola-Mobile-Sales-Analysis-Power-BI-Dashboard
This project presents an interactive Power BI dashboard created using Motorola Mobile Sales data. The goal of this project is to analyse sales trends, customer behaviour, product performance, and payment insights to support data-driven decision-making.

> **I performed end-to-end BI development, including:**
>- Data understanding & requirement gathering
>- Cleaning and transformation using Power Query
>- Creating calculated columns & DAX measures
>- Designing an interactive, user-friendly dashboard
>- Extracting insights for business decisions*
